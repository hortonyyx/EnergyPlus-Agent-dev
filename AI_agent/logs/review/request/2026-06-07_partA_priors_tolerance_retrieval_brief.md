## 0. 背景（够用即可）

partA = 识图建模管线的"容差校正层"：把图纸感知的噪声/矛盾，在容差内修成 EP 友好的干净几何，并记录每次修正。其中两类内容需要**权威专业数值**，本任务负责检索汇编：

- **A4 建筑常识先验**：当图纸尺寸缺失/矛盾/低置信时，用作裁决兜底的"合理值"（如立面没标窗尺寸链时按常规窗台/窗高配置；挡掉把窗宽当房宽造出的"幽灵房"）。
- **A0 容差常数**：校正用的几个阈值需要权威出处而非拍脑袋。

**红线（务必随数据带出）**：先验是**裁决/兜底、不得覆盖一致测量数据**；必须**按建筑类型 + 空间类型分型**，禁止全局单一值；每个值带"典型值 + 合理区间 + 出处 + 权威级别"。

## 1. Part A — A4 建筑常识先验表（重点）

**起步建筑类型 = 办公建筑**（当前测试语料是办公楼）；可顺带给"通用/住宅"列作对照，但办公优先、字段必填。

### 1.1 需要的表（每个值给：典型值 / 合理区间 [min,max] / 出处+条款 / 权威级别 / 适用建筑·空间类型 / 备注）

1. **门窗洞口尺寸**
   - 外窗：窗台高（sill height）、窗高（head/height）、单窗宽常见档
   - 门：门宽（户门/房间门/疏散门）、门高
2. **建筑模数 / 轴网**
   - 基本模数、常用柱网/开间进深模数（如 3M/6M、办公常见柱网 7.2/8.1/8.4m 等）
3. **各空间类型房间最小/典型尺寸**（按 space type 分行）：
   - office（单间/开敞）、corridor（单/双走道净宽）、WC（厕位/隔间）、stair（楼梯间/梯段净宽）、lift（电梯井/候梯厅）、storage、lobby、MEP/机房、茶水/休息
   - 字段：最小净宽 / 最小净深 / 最小净面积 / 典型层高相关项
4. **层高 / 净高**
   - 办公层高常见范围、净高下限、走道净高下限
5. **WWR / 立面**（与节能规范挂钩）
   - 各朝向 WWR 常见/上限（公共建筑节能口径）

### 1.2 权威级别排序（务必标注每条来自哪一级）

1. **中国国家/行业规范（最高）**：GB 50352《民用建筑设计统一标准》、GB 50189《公共建筑节能设计标准》、JGJ 67《办公建筑设计标准》、GB 50016《建筑设计防火规范》（疏散宽度/门）、GB 50096《住宅设计规范》（住宅对照）、GB/T 50002《建筑模数协调标准》等——**给具体条款号**。
2. **国际权威工具书/标准（交叉校核）**：Neufert《Architects' Data》、ASHRAE 90.1（含 Appendix G 周边进深 4.6m/15ft）、IBC、OpenStudio-standards 几何约定。
3. 行业惯例（最低，仅在前两级缺失时用，明确标"惯例无强制出处"）。

冲突时以国标为准、注明与国际值的差异。

## 2. Part B — A0 容差常数权威化（轻）

验证/修正下列常数，各给"建议值 + 依据 + 出处"：

| 常数 | 当前拟用值（待证） | 要回答 |
|---|---|---|
| 坐标吸附栅格 | 50 mm | BEM/方案阶段是否合理；有无规范/工具默认 |
| 最小边长（退化面阈值） | 0.1 m | EnergyPlus 对退化面的实际容忍/崩溃边界 |
| 闭缝阈值（gap close） | 待定（曾议 500 mm） | 多大缝判"应闭合"合理、与墙厚关系 |
| 面积 / WWR 相对容差 | ±5% | 是否对齐常见 BEM 校核口径 |
| 周边进深（perimeter depth） | 4.6 m / 15 ft | 确认 ASHRAE 出处 + 中国实践是否不同 |
| 坐标量化粒度 | 10 mm（方案）/ 50 mm（仿真） | 是否合理 |

## 3. 输出格式

- 结构化表（Markdown 表或 JSON 均可），**可直接 drop 进 A4 / 回填 A0**。
- 每个数值条目字段：`value_typical` / `range[min,max]` / `unit` / `building_type` / `space_type` / `source`（规范名+条款号 或 文献+页）/ `authority`（national_code / international / convention）/ `note`。
- 末尾附"红线落实说明"：哪些值适合作硬下限、哪些只宜作 warning/score（呼应 A4 先验只出 score 不直接改的门控）。
