# Prototype

This directory contains the final EnergyPlus Agent showcase animation.

## Open

Open `index.html` directly in a browser.

## Runtime

- GSAP CDN drives the main animation timelines.
- `script.js` contains playback state, slide mode, continuous mode, and per-shot GSAP
  timelines.
- `styles.css` contains static layout, colors, cards, SVG styling, and fallback CSS
  animations.

## Navigation

- Left directory: play the selected shot from its start.
- Right directory: jump directly to the selected shot's final frame.
